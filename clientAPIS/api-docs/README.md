# API Documentation

Este proyecto contiene la documentación de la API en varios formatos para facilitar su uso y prueba.

## Estructura del Proyecto

- **insomnia/endpoints.json**: Contiene la representación JSON de todos los endpoints de la API, formateada para su uso en Insomnia 11.0.2. Incluye detalles como métodos de solicitud, URLs, encabezados y parámetros del cuerpo.

- **openapi/swagger.yaml**: Documento YAML que describe la API utilizando la Especificación OpenAPI. Incluye información sobre los endpoints de la API, formatos de solicitud/respuesta y métodos de autenticación.

- **postman/collection.json**: Contiene una colección de Postman en formato JSON, que permite a los usuarios importar los endpoints de la API en Postman para pruebas y propósitos de documentación.

## Instrucciones de Uso

1. **Insomnia**: Importa el archivo `endpoints.json` en Insomnia para comenzar a probar los endpoints de la API.
2. **OpenAPI**: Utiliza el archivo `swagger.yaml` para generar documentación interactiva o para integraciones automáticas.
3. **Postman**: Importa la colección desde `collection.json` en Postman para acceder a los endpoints de la API.

Asegúrate de revisar cada archivo para obtener detalles específicos sobre los endpoints y su uso.
